*该项目采取模块化编程方式，主要包含5个文件：
*除本说明文件外，还有
*主函数文件（main.cpp）、头文件（statement.h）、创建素数数组文件（creatPrimeArray.cpp）、创建输入整数的数组文件（creatInputNum.cpp）
*本项目除实现题目基本要求外，还有如下亮点：
*1、为避免每次验证时，都需要做是否为素数判断（循环，严重耗时），这里采用“空间换时间”的方式，将素数预先存入数组中
*2、获得素数数组，我们采用“筛选法”，速度更快
*3、实现用户一次性输入多个整数（以0为结束），一次性输出验证结果
*4、验证过程中，输入不合理或导致异常，会抛出错误，并给出错误信息，且程序不会终止
*5、实现用户友好界面，用户可以选择输出一种素数分解方法，或所有素数分解方法
